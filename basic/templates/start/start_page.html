Приложение DATA-Y
поиск и анализ аномалий в начислениях за тепловую энергию

Приложение используется для анализа отчетов потребления тепловой энергии с целью выявления случаев ошибочно внесённых показаний, а также случаев аномально низкого потребления тепловой энергии.

Данная система анализирует данные о потреблении тепловой энергии и выявляет аномальные начисления с использованием модели искусственного интеллекта (ИИ).

Состав разделов приложения:
Справочник по объектам потребления (Объекты)

представленная база данных объектов

база данных объектов, дополненная объектами учета потребления, которых не было в предоставленной базе

Данные по потреблению тепловой энергии

данные по потреблению за отчетный месяц

объекты с нулевыми значениями показаний за тепловую энергию за отчетный период

данные по потреблению за ВСЕ периоды

объекты с нулевыми значениями показаний за тепловую энергию за ВСЕ периоды

потребление многоквартирных домов с индикатором отклонения от медианы более чем на 25% (за каждый период) - ВСЕ периоды

Анализ данных по потреблению

равные значения показаний в течение нескольких расчетных периодов

выборка данных по объекту за ВСЕ периоды

выбор нулевых данных за заданный период

объекты без данных по потреблению (за ВСЕ периоды)

потребление многоквартирных домов с индикатором отклонения от медианы более чем на 25% (за заданный период)

Результаты анализа данных по потреблению с использованием ML

индикаторы аномалий всех объектов за каждый период

многоквартирные дома с аномальным потреблением за месяц с учетом PCA коэффициентов.

отклонения от значений предсказанных эталонной моделью

выборка объектов с аномалиями по заданным условиям

Сервис
(доступен только администратору приложения)
Загрузка данных:
  • загрузить данные потребления тепловой энергии за отчетнай месяц;
  • загрузить данные об объектах потребления тепловой энергии;
  • загрузить результаты расчета ML-моделью индикаторов аномальности потребления за месяц;
  • загрузить результаты расчета ML-модели прогноза потребления за период
Расчет аномальных данных:
  • пересчитать равные значения показаний по периодам;
  • порядок расчета аномального потребления по объектам:
    1. добавить в базу объектов новые объекты / объекты из отчетов по потреблению
    2. добавить данные объектов (этажность, год постройки, площадь) в данные по потреблению
      3. Для подготовки данных и рассчета аномально низкого/высокого потребления выполнить:
    3. создать группы объектов (apartment_object_group)
    4. рассчитать границы аномального потребления за месяц в разрезе групп.
Прочие сервисы:
  • вывести данные по различным отчетам в файл .xlsx ( доработка при необходимости)
